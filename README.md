# Sales Analysis Project

## Project Overview

This project focuses on analyzing sales data using MySQL, Python (Pandas, Matplotlib, Seaborn), and Power BI.

The goal of this project is to identify business insights such as:

- Highest revenue generating category
- Most profitable category
- Top customers by sales
- State-wise sales performance
- Sub-category performance
- Monthly sales trends
- Sales target comparison

This project helps demonstrate practical Data Analyst skills including data cleaning, SQL querying, data visualization, and dashboard creation.

## Dataset Used

The project uses 3 CSV files:

1. List of Orders.csv
2. Order Details.csv
3. Sales Target.csv

These datasets contain customer details, order information, sales amount, profit, quantity, category, and monthly sales targets.

## Key Insights

- Electronics generated the highest sales
- Furniture showed strong profit contribution
- Top customers contributed major revenue share
- Some states performed significantly better than others
- Seasonal trends were visible in monthly sales
